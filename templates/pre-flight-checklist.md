# Pre-Flight Checklist (Template)

> **Gate before writing.** The conductor works through this list **before** the Writer begins a chapter. Purpose: pre-empt the most common failure sources (late canon reversals, drifting values, unprotected passages, missing voice signature). Lives at `templates/pre-flight-checklist.md`; run per chapter.

## Story-Level Pre-Flight (run ONCE, before Chapter 1)

> **Hard-won lesson:** discovering the ending and midpoint *while drafting* forces backfilling and rewrites. Plan the whole arc first.

- [ ] **Arc plan exists.** `story/story-arc-plan.md` is filled in (from `templates/story-arc-plan.md`).
- [ ] **Ending is decided**, not just gestured at (Arc Plan §2).
- [ ] **Load-bearing decisions all ≥ 🟡** in `canon/canon-decision-register.md` (Arc Plan §6: ending, cost, core rule + limits, antagonist's true nature, protagonist arc, midpoint, reveal ladder, scope).
- [ ] **Protagonist arc is premise-native** (would not slot unchanged into a generic story).
- [ ] **Owed-payoff ledger started** (Arc Plan §5, mirrored to `canon/setup-payoff-register.md`).
- [ ] **Author has signed off** on the arc (Arc Plan §8).

> Any box unticked → **do not draft Chapter 1.** Finish planning first.

## Checklist — Chapter {{N}} ({{POV}})

- [ ] **Gating canon resolved.** No entry with status 🔴 in `canon/canon-decision-register.md` names this chapter as dependent. (Otherwise: have the author decide first.)
- [ ] **Voice signature loaded.** The POV character's line from `style/voice-signature.md` handed to the Writer.
- [ ] **Tic blocklist handed over.** The current `style/tic-blocklist.md` in the Writer brief.
- [ ] **Values checked.** All cross-chapter numbers/names reconciled against `canon/conventions.md` + `canon/facts-register.md`; no divergence open.
- [ ] **Diegesis protection in hand.** Relevant protected passages from `open-points/diegesis-protection-register.md` handed to the Writer.
- [ ] **Open items reviewed.** Bookkeeping entries touching this chapter pulled into the brief.
- [ ] **Scene goal + T/C target set.** The chapter frontmatter (`templates/chapter.md`) filled in.

> All boxes ticked → set `gating_checked: true` in the chapter frontmatter, start the Writer. An open box on gating canon → **pause the pipeline, ask the author.**
